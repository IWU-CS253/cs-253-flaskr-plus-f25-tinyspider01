# Flask Example

An example Flask application (adapted from the [Flask 0.12 tutorial](https://www.iwu.edu/~mliffito/flask_tutorial/))

## Description:
In the project we were tasked with rebranding flaskr to
flaskr plus. One attribute was added called Category.
With Category, there are 2 functions added as well. The sort
and delete functions. The Sort function sorts by the category.
The delete function deletes the post.
